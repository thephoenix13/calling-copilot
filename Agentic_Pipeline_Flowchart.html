Step Gate Conditions

Gate 1→2
A job must be selected (job_id is set on the session)
Gate 2→3
JD assets generated and saved (enhancement_saved = 1)
Gate 3→4
At least one candidate added to the session
Gate 4→5
At least one candidate marked Shortlisted (screening_status = pass)
Gate 5→6
Optional step — can skip if no video interviews needed
Gate 6→7
All shortlisted candidates reviewed; all "Move Ahead" have a score
Gate 7→8
At least one candidate set to Proceed with an interview level
POFU Entry
Candidate reaches Selected column in Kanban → recruiter enrols manually
